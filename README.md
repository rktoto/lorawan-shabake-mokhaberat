# LoRaWAN Simulation — MATLAB

خلاصه
- این پروژه یک شبیه‌ساز LoRaWAN پیاده‌سازی شده در MATLAB است که تاثیر تعداد نودها و Spreading Factor (SF) را بر نسبت تحویل بسته (PDR) و مصرف انرژی بررسی می‌کند.
- کدها به‌صورت ماژولار نوشته شده‌اند: اسکریپت اصلی `sim_main.m` و توابع کمکی در پوشه `functions/`.

نیازمندی‌ها
- MATLAB (توصیه‌شده: R2018a یا بالاتر)
- نداردِ کتابخانهٔ خارجی لازم است (در صورت نیاز در docs/server_setup.md ذکر می‌شود)

ساختار پروژه
- sim_main.m : اسکریپت اصلی شبیه‌سازی
- functions/ : توابع کمکی
- results/ : نمودارها و خروجی‌ها (png, mat, csv)
- docs/ : گزارش و دیاگرام‌ها
- data/ : فایل‌های ورودی (اختیاری)

نحوه اجرا
1. پوشه پروژه را در MATLAB باز کنید:
   ```matlab
   cd /path/to/project
   ```
2. اسکریپت اصلی را اجرا کنید:
   ```matlab
   run sim_main.m
   ```
3. نتایج در پوشه `results/` ذخیره می‌شوند.

نکات
- تمام پارامترهای شبیه‌سازی در بالای `sim_main.m` قرار دارند؛ برای بازتولید دقیق نتایج، آن‌ها را ثبت یا در فایل پارامترها قرار دهید.
- برای تحویل کلاسی: یک release با برچسب `v1.0` بسازید و `report.pdf` را در release آپلود کنید.
